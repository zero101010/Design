# Design

Repositório de aprendizado e portfólio em design gráfico. Contém materiais de estudo organizados por tema e projetos práticos com briefing detalhado.

---

## Estudo

### Fundamentos

| Tópico | Descrição |
|--------|-----------|
| [Hierarquia Visual](study/fundamentos/hierarquia-visual.md) | Como guiar o olhar do espectador — tamanho, contraste, posição, whitespace |
| [Teoria das Cores](study/fundamentos/teoria-das-cores.md) | Círculo cromático, harmonias, psicologia das cores, RGB vs. CMYK |
| [Tipografia](study/fundamentos/tipografia.md) | Anatomia, classificação, combinação de fontes, legibilidade, fontes gratuitas |
| [Grid e Espaçamento](study/fundamentos/grid-e-espacamento.md) | Sistemas de grid, regra dos terços, whitespace, proporção áurea |
| [Gestalt](study/fundamentos/gestalt.md) | 8 princípios da percepção visual com exemplos de logos reais |

### Ferramentas

| Ferramenta | Descrição |
|------------|-----------|
| [Figma](study/ferramentas/figma.md) | UI/UX, prototipagem, layouts digitais — gratuito para uso individual |
| [Adobe Illustrator](study/ferramentas/illustrator.md) | Vetores, logos, ícones, identidade visual — padrão da indústria |
| [Adobe Photoshop](study/ferramentas/photoshop.md) | Edição de imagem, composições, mockups fotorrealistas |
| [Adobe InDesign](study/ferramentas/indesign.md) | Diagramação editorial — revistas, livros, zines, cardápios |
| [Ferramentas Gratuitas](study/ferramentas/ferramentas-gratuitas.md) | Figma, Photopea, Inkscape, Canva e recursos sem custo |
