# Adobe Photoshop

Photoshop é o software padrão para edição de imagens raster (baseadas em pixels). É usado para retocar fotos, criar composições, manipular imagens, criar mockups e produzir texturas. No contexto do design gráfico, é mais uma ferramenta de suporte do que de criação — a identidade visual nasce no Illustrator ou Figma, mas o Photoshop entra quando há fotos e manipulações de imagem envolvidas.

---

## Quando Usar Photoshop

- Retocar ou ajustar fotos (brilho, contraste, saturação, remoção de elementos)
- Criar composições com múltiplas fotos
- Aplicar logos e peças em mockups fotorrealistas
- Criar texturas, fundos e efeitos especiais
- Editar imagens para redes sociais com elementos fotográficos
- Criar capas com fotografia

## Conceitos Fundamentais

### Pixels e Resolução
- **72 PPI (pixels per inch):** padrão para telas/web. Arquivos menores.
- **150 PPI:** aceitável para impressão simples
- **300 PPI:** padrão mínimo para impressão profissional
- Sempre crie documentos em 300 DPI se houver qualquer chance de impressão

### Layers (Camadas)
O conceito mais importante do Photoshop. Cada elemento fica em uma camada separada, permitindo editar partes sem afetar o resto. Organize as camadas com nomes e grupos desde o início.

### Layer Masks (Máscaras de Camada)
Mascaram (ocultam) partes de uma camada sem apagar — o que foi ocultado pode ser recuperado a qualquer momento. É a forma correta de recortar elementos no Photoshop (nunca use borracha em camadas editáveis).

**Como funciona:**
- Preto na máscara = oculta
- Branco na máscara = mostra
- Cinza = semi-transparente

### Adjustment Layers (Camadas de Ajuste)
Aplicam correções de cor e tom de forma não-destrutiva — o ajuste fica numa camada separada e pode ser editado ou removido a qualquer momento. Nunca faça ajustes diretamente na camada de imagem.

**Ajustes mais usados:**
- **Curves:** controle preciso de brilho e contraste por canal
- **Hue/Saturation:** altera matiz, saturação e luminosidade
- **Color Balance:** ajusta a temperatura de cor por sombras, médios e altas luzes
- **Levels:** controla pontos negros, brancos e médios-tons

### Blend Modes (Modos de Mesclagem)
Define como uma camada interage com as camadas abaixo. Os mais usados no design:
- **Multiply:** escurece, remove brancos (bom para texturas sobre cores)
- **Screen:** clareia, remove pretos
- **Overlay:** aumenta contraste e saturação
- **Soft Light:** versão suave do Overlay

### Smart Objects
Converte uma camada em um objeto inteligente. Permite redimensionar sem perda de qualidade e aplicar filtros de forma não-destrutiva. **Sempre converta imagens em Smart Objects antes de transformá-las.**

## Roteiro de Aprendizado

**Semana 1 — Interface e ferramentas básicas**
- [ ] Criar documentos com resolução correta (72 PPI para digital, 300 PPI para impressão)
- [ ] Entender o painel de Layers
- [ ] Crop Tool e Transform (⌘T / Ctrl+T)
- [ ] Selection Tools: Rectangular, Lasso, Quick Selection

**Semana 2 — Seleção e máscaras**
- [ ] Select Subject (IA integrada — seleciona o sujeito automaticamente)
- [ ] Quick Mask mode para refinar seleções
- [ ] Layer Masks: criar, pintar, refinar
- [ ] Select and Mask para recortes de cabelo e elementos complexos

**Semana 3 — Correção de imagem**
- [ ] Adjustment Layers: Curves, Levels, Hue/Saturation, Color Balance
- [ ] Camera Raw Filter para edição fotográfica completa
- [ ] Clone Stamp Tool e Healing Brush para retoques
- [ ] Content-Aware Fill para remoção de elementos

**Semana 4 — Composição e efeitos**
- [ ] Smart Objects e transformações não-destrutivas
- [ ] Blend Modes para composição
- [ ] Layer Styles: Drop Shadow, Stroke, Glow, Bevel
- [ ] Filters: Blur, Sharpen, Distort

**Semana 5 — Aplicação no design gráfico**
- [ ] Criar mockups: colocar logo em camiseta, sacola, celular
- [ ] Criar composições para capas (livro, álbum)
- [ ] Exportar para web: File > Export > Export As (PNG, JPG, WebP)
- [ ] Exportar para impressão: PDF com 300 DPI e perfil de cor correto

## Mockups no Photoshop

Mockup é uma apresentação realista de um projeto em contexto — um logo aplicado em uma xícara, um cartaz em uma parede, um app em uma tela de iPhone.

**Como usar um template de mockup:**
1. Baixe um arquivo `.psd` de mockup (Freepik, Mockup World)
2. Abra no Photoshop
3. Localize a camada com um ícone de Smart Object (ícone de página dobrada)
4. Dê duplo clique na camada → abre um arquivo separado
5. Cole sua arte nesse arquivo, salve (⌘S)
6. Volte ao mockup — sua arte já aparece aplicada

## Projeto de Prática Sugerido

Crie uma composição para uma capa de álbum ou cartaz usando:
- Uma foto de fundo (pode ser do Unsplash, que é gratuito)
- Um elemento sobreposto com Blend Mode
- Uma adjustment layer de cor para criar uma identidade visual
- Tipografia (texto sobre a imagem)
- Exporte em 300 DPI para impressão e 72 PPI para digital
