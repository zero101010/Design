# Figma

Figma é a principal ferramenta de design de interfaces e prototipagem usada no mercado atualmente. É **gratuita** para projetos individuais, roda no navegador (sem instalação), e é a ferramenta padrão em empresas de tecnologia e agências digitais. Mesmo que você queira trabalhar com identidade visual, saber Figma é um diferencial enorme.

---

## Por que Figma?

- Roda no navegador — sem custo de licença para uso individual
- Colaboração em tempo real (como Google Docs, mas para design)
- Tem substituído parcialmente o Illustrator e o Sketch no mercado digital
- Comunidade enorme: templates, ícones, componentes gratuitos na plataforma
- Atalhos e lógica parecidos com Illustrator — aprender um facilita o outro

## Conceitos Fundamentais

### Frames vs. Artboards
No Figma, você trabalha dentro de **Frames** (equivalente ao artboard do Illustrator). Todo projeto começa criando um Frame no tamanho certo (iPhone, Desktop, A4, etc.).

### Auto Layout
O recurso mais poderoso do Figma. Permite criar componentes que se ajustam automaticamente quando o conteúdo muda — como um flexbox visual. Essencial para design de interfaces.

### Components e Variants
Um componente é um elemento reutilizável (botão, card, ícone). Edite o componente mestre e todas as instâncias atualizam automaticamente. Variants são estados diferentes do mesmo componente (botão ativo/inativo, hover/default).

### Styles
Salve cores, textos e efeitos como estilos globais. Altere uma cor em um estilo e ela muda em todo o projeto.

### Grids e Constraints
Configure grids de colunas ou linhas em qualquer frame. Constraints definem como um elemento se comporta quando o frame é redimensionado.

## Roteiro de Aprendizado (do básico ao avançado)

**Semana 1 — Interface e navegação**
- [ ] Criar uma conta gratuita em `figma.com`
- [ ] Entender a interface: painel esquerdo (layers), central (canvas), direito (propriedades)
- [ ] Criar formas básicas (retângulo, círculo, linha, polígono)
- [ ] Trabalhar com cores de preenchimento e borda
- [ ] Usar texto e estilos de texto

**Semana 2 — Composição**
- [ ] Criar e organizar Frames
- [ ] Alinhar e distribuir elementos
- [ ] Usar grupos e frames como contêineres
- [ ] Importar imagens
- [ ] Aplicar grids de colunas

**Semana 3 — Ferramentas avançadas**
- [ ] Boolean operations (Union, Subtract, Intersect, Exclude) — equivalente ao Pathfinder do Illustrator
- [ ] Pen tool (criação de formas vetoriais livres)
- [ ] Máscaras
- [ ] Effects (sombra, blur, inner shadow)

**Semana 4 — Sistema de design**
- [ ] Criar e usar Components
- [ ] Criar e usar Styles (cores, tipografia)
- [ ] Auto Layout básico
- [ ] Variants (estados de componentes)

**Semana 5+ — Prototipagem**
- [ ] Criar conexões entre frames (prototype mode)
- [ ] Adicionar transições e animações
- [ ] Smart Animate
- [ ] Apresentar protótipo para teste

## Recursos Gratuitos da Comunidade Figma

- **Figma Community** (`figma.com/community`) — templates, ícones, UI kits; tudo gratuito
- **Material Design 3 Kit** — UI kit do Google disponível na Community
- **Apple Design Resources** — componentes iOS disponíveis na Community
- **Iconify plugin** — biblioteca com mais de 100.000 ícones gratuitos dentro do Figma

## Projeto de Prática Sugerido

Crie um wireframe de 3 telas de um aplicativo fictício (ex: app de receitas):
1. Tela de login
2. Feed de receitas
3. Detalhe de uma receita

Use: Frame, texto, cores, imagens de placeholder (rectangles coloridos), e ao final adicione uma prototipagem clicável entre as telas.
