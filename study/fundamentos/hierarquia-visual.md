# Hierarquia Visual

A hierarquia visual é o princípio que orienta o olhar do espectador pela composição em uma ordem intencional — do elemento mais importante ao menos importante. É o fundamento que torna uma peça gráfica legível e comunicativa.

---

## Conceitos-chave

### O que é hierarquia visual?
É a organização dos elementos visuais de forma que o cérebro os processe em uma sequência lógica. Sem hierarquia, todos os elementos competem igualmente pela atenção, causando confusão.

### Como criar hierarquia?

**Tamanho**
O elemento maior atrai mais atenção. Um título grande domina a página antes de qualquer outro elemento.

**Contraste**
Diferenças de cor, peso e brilho criam distinção. Um botão escuro em fundo claro naturalmente chama mais atenção.

**Cor**
Cores saturadas e quentes avançam visualmente; cores frias e dessaturadas recuam. Use cor para destacar o elemento principal.

**Posicionamento**
O canto superior esquerdo é lido primeiro (em culturas ocidentais, que leem da esquerda para direita, de cima para baixo). O centro geométrico também captura atenção.

**Espaço em branco (whitespace)**
Isolar um elemento com espaço ao redor aumenta sua importância. O vazio não é ausência — é um instrumento de destaque.

**Tipografia**
Variação de peso (bold vs. regular), tamanho e família cria camadas de informação: título → subtítulo → corpo → rodapé.

**Repetição e consistência**
Elementos que se repetem com o mesmo estilo são percebidos como do mesmo nível hierárquico.

---

## Os 3 níveis de hierarquia

| Nível | Função | Exemplo |
|-------|--------|---------|
| 1º — Entrada | Capturar a atenção imediatamente | Título principal, imagem hero |
| 2º — Orientação | Confirmar o interesse e direcionar | Subtítulo, chamada secundária |
| 3º — Detalhe | Entregar a informação completa | Corpo de texto, rodapé, informações de contato |

## Exercício prático

Pegue qualquer cartaz ou capa de revista e responda:
1. O que você vê primeiro? Por quê?
2. Qual é a segunda coisa que você vê?
3. Existe algum elemento que compete desnecessariamente com o principal?
4. Se você tivesse 3 segundos olhando para essa peça, qual informação conseguiria capturar?

## Conexão com outros tópicos
- **Tipografia** → os pesos e tamanhos de fonte são a principal ferramenta de hierarquia em texto
- **Teoria das Cores** → contraste cromático reforça ou quebra a hierarquia
- **Grid** → a posição dos elementos no grid define onde o olho vai primeiro
