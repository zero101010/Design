# Tipografia

Tipografia é a arte de organizar o texto de forma visualmente eficaz. Mais de 90% do design é tipografia — a escolha da fonte, o espaçamento entre letras, a entrelinha e o tamanho do texto determinam se uma peça é lida ou ignorada.

---

## Anatomia de uma Fonte

Conhecer a terminologia ajuda a falar sobre tipografia com precisão:

| Termo | Definição |
|-------|-----------|
| **Baseline** | Linha invisível onde as letras se apoiam |
| **Cap height** | Altura das letras maiúsculas |
| **x-height** | Altura das letras minúsculas (sem ascendentes) |
| **Ascendente** | Parte da letra que sobe acima da x-height (b, d, h, l) |
| **Descendente** | Parte da letra que desce abaixo da baseline (g, p, y) |
| **Serifa** | Os "pés" nas extremidades das letras |
| **Kerning** | Espaço entre dois caracteres específicos |
| **Tracking** | Espaço uniforme entre todos os caracteres de um texto |
| **Leading** | Espaço entre linhas (entrelinha) |

## Classificação das Fontes

### Serif (Com serifa)
Possuem "pés" nas extremidades. Transmitem tradição, elegância, confiança, autoridade.
- **Usos:** textos longos impressos (livros, jornais), marcas institucionais, moda de luxo
- **Exemplos:** Times New Roman, Garamond, Georgia, Playfair Display

### Sans-serif (Sem serifa)
Sem "pés". Transmitem modernidade, limpeza, neutralidade, acessibilidade.
- **Usos:** interfaces digitais, startups, textos em tela, design minimalista
- **Exemplos:** Helvetica, Futura, Inter, Open Sans, Montserrat

### Monospace
Cada caractere ocupa exatamente a mesma largura. Associada a código e máquinas de escrever.
- **Usos:** projetos tecnológicos, textos de código, design retrô
- **Exemplos:** Courier, IBM Plex Mono, JetBrains Mono

### Display / Decorativa
Projetadas para grandes tamanhos, não para texto corrido. Alta personalidade.
- **Usos:** títulos, logos, pôsteres — nunca em corpo de texto
- **Exemplos:** Bebas Neue, Abril Fatface, fontes de caligrafia

### Script / Manuscrita
Imitam caligrafia ou escrita à mão. Elegantes, pessoais, femininas ou artesanais dependendo da escolha.
- **Usos:** convites, embalagens artesanais, marcas de beleza — com moderação
- **Exemplos:** Pacifico, Great Vibes, Lobster

## Combinando Fontes (Type Pairing)

A regra geral: **contraste sem conflito**.

**Combinações que funcionam:**
- Serif para título + Sans-serif para corpo (contraste clássico)
- Display para título + Sans-serif simples para corpo
- Duas fontes da mesma família tipográfica em pesos diferentes

**Regras práticas:**
1. Use no máximo **2-3 fontes** em um projeto
2. Evite combinar duas fontes com personalidades semelhantes (duas display juntas, por exemplo)
3. Estabeleça hierarquia clara: qual é a fonte principal, qual é a de suporte?
4. Teste a combinação em diferentes tamanhos antes de decidir

**Recursos para testar combinações:**
- **Google Fonts** (`fonts.google.com`) — clique em qualquer fonte e veja sugestões de combinação em "Pairings"
- **Fontpair** (`fontpair.co`) — combinações curadas especificamente para Google Fonts
- **Typewolf** (`typewolf.com`) — exemplos reais de combinações tipográficas em sites

## Escala Tipográfica

Em vez de escolher tamanhos aleatoriamente, use uma escala proporcional:

**Escala modular** — cada tamanho é multiplicado por um fator fixo. Exemplo com fator 1.25:
- 12px → 15px → 18.75px → 23.4px → 29.3px...

**Ferramenta:** `type-scale.com` — gere escalas tipográficas automaticamente

## Legibilidade e Leiturabilidade

**Legibilidade:** se cada letra individual é fácil de distinguir
**Leiturabilidade:** se o texto corrido é fácil de ler

**Configurações para boa leiturabilidade em texto corrido:**
- Leading (entrelinha): 120% a 150% do tamanho da fonte é confortável
- Tracking: neutro ou ligeiramente positivo para corpo de texto
- Largura de coluna: 45-75 caracteres por linha é o ideal para leitura confortável
- Tamanho mínimo: 16px para texto em tela, 9pt para impressão

## Exercício Prático

1. Escolha um texto curto (uma frase de impacto + 2-3 linhas de descrição)
2. Componha o mesmo texto com 3 combinações tipográficas diferentes no Canva ou Figma:
   - Versão séria e institucional
   - Versão moderna e minimalista
   - Versão expressiva e criativa
3. Avalie: qual transmite melhor a mensagem? Por quê?
