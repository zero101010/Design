# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Contexto do Repositório

Este repositório é um espaço de aprendizado pessoal de **design gráfico** para uma estudante recém-formada, sem experiência prévia, que deseja construir portfólio e adquirir conhecimentos práticos na área.

**Objetivo:** Documentar o progresso de aprendizado, organizar projetos práticos e construir um portfólio.

---

## Roadmap de Aprendizado

### Fase 1 — Fundamentos Teóricos

**Princípios de Design**
- Hierarquia visual
- Teoria das cores (círculo cromático, paletas, contraste)
- Tipografia (famílias tipográficas, combinações, legibilidade)
- Grid e espaçamento (regra dos terços, espaço em branco)
- Gestalt (proximidade, semelhança, continuidade, figura-fundo)

**Referências para estudo:**
- *Thinking with Type* — Ellen Lupton
- *The Elements of Typographic Style* — Robert Bringhurst
- *Logo Design Love* — David Airey
- Canais no YouTube: The Futur, Satori Graphics, Will Paterson

---

### Fase 2 — Ferramentas Essenciais

**Vetor e identidade visual**
- **Adobe Illustrator** — criação de logos, ícones, ilustrações vetoriais
- **Figma** (gratuito) — UI/UX, prototipagem, design de interfaces

**Edição de imagem**
- **Adobe Photoshop** — manipulação de fotos, composições, mockups
- **Affinity Photo** (alternativa paga única, sem assinatura)

**Layout editorial**
- **Adobe InDesign** — revistas, livros, catálogos, materiais impressos

**Ferramentas gratuitas para começar:**
- **Figma** (free tier) — substitui boa parte do Illustrator para digital
- **Canva Pro** / **Adobe Express** — para projetos rápidos
- **GIMP** — alternativa gratuita ao Photoshop
- **Inkscape** — alternativa gratuita ao Illustrator

---

### Fase 3 — Projetos Práticos para Portfólio

Cada projeto deve ter: briefing fictício, processo (sketches/rascunhos), resultado final e apresentação em mockup.

#### Projetos de Identidade Visual
1. **Logo para negócio local fictício** — cafeteria, barbearia, livraria
2. **Redesign de marca existente** — propor nova versão com justificativa
3. **Mini brand identity** — logo + paleta + tipografia + cartão de visita

#### Projetos de Material Gráfico
4. **Cartaz/Poster** — evento cultural, show, exposição fictícia
5. **Capa de álbum** — escolher gênero musical e criar conceito visual
6. **Embalagem** — produto simples (café, sabonete artesanal)

#### Projetos Digitais
7. **Interface de app** — tela inicial e 2-3 telas internas no Figma
8. **Infográfico** — dados de tema de interesse pessoal
9. **Feed de Instagram** — 9 posts coesos para uma marca fictícia

#### Projetos Editoriais
10. **Zine de 8 páginas** — tema livre, layout com tipografia e imagem
11. **Cardápio de restaurante** — impresso e versão digital

---

### Fase 4 — Apresentação do Portfólio

**Plataformas recomendadas:**
- **Behance** (Adobe) — padrão da indústria para designers
- **Dribbble** — foco em UI e identidade visual
- **Notion** — portfólio simples e rápido de montar

**Estrutura de cada case no portfólio:**
1. Briefing / problema a resolver
2. Processo e decisões criativas
3. Resultado final em mockups realistas
4. Aprendizados

**Recursos para mockups gratuitos:**
- Mockup World
- Freepik (mockups gratuitos)
- ls.graphics

## Notas para o Claude Code neste Repositório

- Os arquivos deste repositório são principalmente documentos (`.md`), imagens e arquivos de design exportados — não há código executável.
- Ao ajudar com projetos, sempre criar um `briefing.md` antes de sugerir soluções visuais.
- Ao sugerir paletas de cores, usar formato HEX e mencionar o contexto de uso (impressão vs. digital → CMYK vs. RGB).
- Ao revisar projetos, avaliar segundo os princípios da Fase 1.
