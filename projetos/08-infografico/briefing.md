# Projeto 08 — Infográfico

## Briefing do Projeto

**Tipo:** Infográfico editorial estático
**Publicação fictícia:** Revista Comum — revista digital independente de jornalismo cultural
**Formato de publicação:** Digital (site e redes sociais)
**Formato do arquivo:** Vertical, largura 1080px (padrão para redes e fácil de embutir em sites)

**Tema do infográfico:**
"Cinema Nacional: Quem Faz os Filmes Brasileiros?" — um panorama sobre a representatividade de gênero e raça na direção de longas-metragens nacionais nos últimos 10 anos.

*(Nota: os dados são fictícios para este exercício. Crie números plausíveis ou pesquise dados reais da ANCINE para tornar o projeto mais rico.)*

---

## Dados para o Infográfico (podem ser adaptados ou pesquisados)

**Produza dados fictícios plausíveis para:**

1. **Participação por gênero na direção**
   - % de filmes dirigidos por homens vs. mulheres vs. não-binário (por ano, nos últimos 10 anos)

2. **Participação por raça/etnia na direção** (dados fictícios)
   - % aproximado de diretores brancos, negros, indígenas, asiático-brasileiros

3. **Comparação com outros países**
   - Cite 4-5 países e o % de filmes dirigidos por mulheres (dados ficcionais)

4. **Uma mudança notável**
   - Ex: "Entre 2018 e 2023, o número de estreantes negras na direção aumentou X%"

5. **Uma citação** — crie uma citação fictícia de uma diretora fictícia para contextualizar os dados

---

## Metas do Projeto

1. Os dados devem ser **compreensíveis em 60 segundos** — quem passa pelo infográfico nas redes sociais precisa capturar a informação principal rapidamente
2. A visualização deve ser **honesta** — nenhum truque visual que distorça os dados (gráficos de pizza em 3D, eixos que não começam no zero, etc.)
3. O design deve ter **identidade editorial** — não deve parecer uma apresentação de PowerPoint com dados jogados
4. A hierarquia deve guiar o olhar: dado principal → contexto → detalhes

---

## Tipos de Visualização Sugeridos

Escolha os mais adequados para cada dado:

| Tipo de dado | Visualização recomendada |
|--------------|--------------------------|
| Proporção (% de homens vs. mulheres) | Gráfico de rosca (donut), barra simples, waffle chart |
| Evolução ao longo do tempo | Gráfico de linha, gráfico de área |
| Comparação entre países | Gráfico de barras horizontais |
| Uma estatística marcante | Grande número isolado com contexto |
| Citação | Bloco de texto destacado tipograficamente |

**Evite:** gráficos 3D, gráficos de pizza com muitas fatias, excesso de cores diferentes por categoria.

---

## Direção Visual Sugerida

**O infográfico tem um ponto de vista editorial** — ele fala sobre sub-representação. O design deve refletir isso sem ser panfletário.

**Paleta sugerida:**
- 1-2 cores de destaque para os dados de representatividade
- Base neutra (fundo branco ou cinza muito claro, texto preto)
- A cor de destaque carrega o dado mais importante

**Tipografia:**
- Hierarquia de 3-4 níveis: título > subtítulo > dado/número > corpo de texto
- Fonte sans-serif para legibilidade na tela em tamanhos variados

**Estilo visual:**
- Sóbrio e editorial — não lúdico, não colorido demais
- O conteúdo é sério; o design deve respeitar isso

## Processo Sugerido

### Etapa 1 — Dados e Roteiro (1-2 dias)
- [ ] Definir ou pesquisar os dados (ANCINE publica relatórios públicos — pesquise no Google)
- [ ] Organizar os dados em ordem de importância: qual é a informação mais importante? Qual é o contexto?
- [ ] Escrever o roteiro do infográfico como se fosse um texto: introdução → dado principal → contexto → detalhe → conclusão

### Etapa 2 — Wireframe (1 dia)
- [ ] Fazer um esboço à mão do layout vertical
- [ ] Decidir: como os blocos de informação se encadeiam de cima para baixo?
- [ ] Decidir: quais dados viram gráfico e quais ficam como texto?

### Etapa 3 — Execução no Illustrator ou Figma (3-5 dias)
- [ ] Criar os gráficos como formas vetoriais (não use gráficos gerados automaticamente pelo Excel/Google Sheets — construa-os manualmente para ter controle visual)
- [ ] Organizar a tipografia com hierarquia clara
- [ ] Garantir espaçamento e ritmo visual adequados

### Etapa 4 — Revisão (1 dia)
- [ ] Alguém consegue entender os dados sem explicação?
- [ ] Os valores dos dados estão corretos nas visualizações?
- [ ] Existe fonte/referência para os dados apresentados?

---

## Ferramentas Sugeridas

- **Adobe Illustrator** — criação dos gráficos como vetores (preferível para controle total)
- **Figma** — alternativa viável para infográficos digitais
- **Datawrapper** (`datawrapper.de`) — ferramenta gratuita que gera gráficos prontos para exportar como SVG (pode ser importado no Illustrator/Figma)

---

## Resultado Esperado

**Arquivos a entregar:**
- Infográfico completo: PNG 1080px de largura, alta qualidade
- Versão para impressão: PDF A3 ou A2, 300 DPI

**Apresentação no portfólio:**
- Contexto: sobre o que é o infográfico e para qual publicação
- Fontes dos dados (mesmo que fictícios, documente)
- Esboços do wireframe
- Resultado final
- Detalhe das visualizações de dados
- Decisões de design: por que este gráfico para este dado?

**Critério de sucesso:**
Mostre o infográfico para alguém por 60 segundos e pergunte quais informações ele conseguiu capturar. Se as 2-3 informações principais foram absorvidas, o design cumpriu seu papel.

---

## Estimativa de Tempo
10 a 15 horas distribuídas em 1 a 2 semanas
