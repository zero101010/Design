# Projeto 10 — Zine de 8 Páginas

## Briefing do Projeto

**Tipo:** Projeto autoral
**Formato:** Zine A5 (14,8x21cm) de 8 páginas — montado com uma única folha A3 dobrada e cortada (método "one-sheet zine")
**Tiragem fictícia:** 100 cópias impressas em risografia (mas o arquivo pode ser impresso em impressora comum)
**Distribuição:** Lojas independentes, eventos, bancas de zine

**Tema:** Completamente livre — este é o projeto mais pessoal do portfólio.

---

## Escolhendo o Tema

O zine é um formato de expressão individual. Funciona melhor quando o tema é algo que você domina ou sobre o qual tem um ponto de vista. Sugestões:

- **Diário visual** de um lugar que você frequenta
- **Guia não-oficial** de algo que você gosta: um bairro, um tipo de comida, uma estação
- **Ensaio sobre um conceito** que te interessa: memória, saudade, barulho, silêncio
- **Entrevista fictícia** com uma figura histórica ou imaginária
- **Coletânea de curiosidades** sobre um tema específico
- **Manifesto pessoal** sobre como você vê o design, a arte ou o mundo

**Evite:** temas genéricos como "natureza" ou "viagem" sem um ponto de vista específico. O melhor zine é o que só você poderia fazer.

---

## Estrutura das 8 Páginas

No zine de 8 páginas com dobragem, a ordem de montagem importa. A estrutura física é:

```
Página 1: Capa
Página 2: Folha de rosto ou primeira página de conteúdo
Página 3: Conteúdo
Página 4: Conteúdo
Página 5: Conteúdo
Página 6: Conteúdo
Página 7: Penúltima página / créditos
Página 8: Contracapa
```

**Para o case do portfólio, foque em:**
- Capa marcante (páginas 1 e 8 são as primeiras a serem vistas)
- Pelo menos 3 páginas de conteúdo com composição forte
- Uma página com texto mais denso para mostrar habilidade de tipografia editorial
- Uma página mais visual/gráfica para mostrar composição

---

## Metas do Projeto

1. Demonstrar **domínio de diagramação** — espaçamentos, grids, hierarquia em página impressa
2. Demonstrar **identidade visual coesa** — as 8 páginas devem parecer uma publicação, não 8 arquivos separados
3. Criar algo com **ponto de vista** — o zine deve parecer autoral, não genérico
4. O arquivo final deve estar **pronto para impressão** — com sangria, margens corretas, CMYK

---

## Considerações de Impressão

**Risografia** (risograph) é um método de impressão alternativo muito popular no universo de zines. Características:
- Impressão em camadas de cores chapadas (cores Pantone)
- Limitações de cores são parte da estética
- Geralmente 2-3 cores por projeto
- Textura ligeiramente granulada, cores sobrepostas criam terceiras cores

**Se você quiser simular a estética risográfica:**
- Trabalhe com 2-3 cores planas (não gradientes complexos)
- Cada cor em uma camada separada no arquivo
- Pense nas cores como transparências — onde se sobrepõem criam efeitos

**Para impressão doméstica/comum:**
- CMYK, qualquer paleta de cores
- Imprima em papel de gramatura levemente maior (90-120g) para melhor resultado

## Processo Sugerido

### Etapa 1 — Conceito e Planejamento (1-2 dias)
- [ ] Escolher o tema e o ponto de vista
- [ ] Escrever/preparar o conteúdo completo — o texto ou os elementos visuais que vão nas 8 páginas
- [ ] Fazer um dummy: imprima uma folha A4, dobre e corte no estilo one-sheet zine para entender a física do objeto
- [ ] Esboçar à mão como cada página vai se parecer (thumbnails)

### Etapa 2 — Identidade do Zine (1 dia)
- [ ] Definir paleta (se risografia: 2-3 cores; se digital: paleta livre)
- [ ] Escolher tipografia: uma fonte para títulos, uma para corpo de texto
- [ ] Definir estilo visual: fotográfico, ilustrado, tipográfico, colagem?

### Etapa 3 — Diagramação no InDesign ou Figma (3-5 dias)
- [ ] Criar documento no InDesign: 8 páginas, A5 (14,8x21cm), 3mm de sangria
- [ ] Criar Master Page com margens e elementos que se repetem
- [ ] Criar Paragraph Styles para todos os níveis de texto
- [ ] Diagramar cada página com atenção ao ritmo visual entre páginas

### Etapa 4 — Preparação para Impressão (1 dia)
- [ ] Preflight: verificar imagens em 300 DPI, fontes incorporadas, cores corretas
- [ ] Exportar PDF para impressão (PDF/X-1a)
- [ ] Se risografia: exportar cada cor em uma camada separada

---

## Ferramentas Sugeridas

- **Adobe InDesign** (ideal) — controle total de tipografia editorial e exportação para impressão
- **Figma** (alternativa viável para quem não tem InDesign)
- **Adobe Illustrator** — se o zine for predominantemente vetorial/ilustrativo
- **Para estética risográfica:** busque no YouTube "risograph effect Photoshop" ou "risograph Illustrator"

---

## Resultado Esperado

**Arquivos a entregar:**
- PDF para impressão (PDF/X-1a, CMYK, 300 DPI, com sangria de 3mm)
- PDF para visualização digital (RGB, sem sangria)
- Se possível: foto física do zine impresso e dobrado

**Apresentação no portfólio:**
- Contexto: qual o tema e o ponto de vista do zine
- Dummy/esboços das páginas
- Resultado final: imagens de cada página abertas
- Mockup do zine físico (Freepik tem mockups de zine/brochura)
- Detalhe de spreads (páginas duplas abertas) mais fortes visualmente
- Se impresso: foto real do objeto

**Critério de sucesso:**
O zine deve parecer um objeto que você quereria pegar e folhear. Mostre fisicamente (impresso ou em pdf scrollável) para alguém e observe se a pessoa se interessa pelo conteúdo além do design.

---

## Estimativa de Tempo
15 a 25 horas distribuídas em 2 a 3 semanas
